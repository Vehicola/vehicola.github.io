What is The Ion Project?

The Ion Project is a project containing Proton, Neutron, Electron and Photon. These are created using Skript 2.2 and skQuery only, I try to stay away from other addons so that more people are willing to use Ion on their servers without the hassle of downloading 5-10 addons.

explore

Features

Ion is a recreation of Essentials, an abandoned project by essentialsteam. My aim is to recreate Essentials within Skript, so that Ion could work on all versions - not just 1.7.

view_carousel

Modules

Currently, Ion is split up into 4 Modules. Proton is the main module, containing over 75% of all commands. Neutron is the chat module, allowing Administartors to customize their chat with groups. Electron is the economy module, built in with Vault. Photon is the settings module, allowing players to modify their server or world with specific settings.
