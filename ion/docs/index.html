Gamemode link

Description: Change a players gamemode to Survival, Creative, Adventure, or Spectator

Permission: proton.gamemode, proton.gamemode.other

  • Usage
  • Example
  • Code

Usage: /gamemode [creative|survival|adventure|spectator] (player)

Aliases: /gm

Example 1: /gamemode creative vexio - Changes Vexio's gamemode to creative

Example 2: /gm s vexio - Changes Vexio's gamemode to survival

Example 3: /gma - Changes the sender's gamemode to adventure
